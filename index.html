HuiChe

基于SpringBoot,QueryDsl 的 Java 快速开发框架

快速上手 →

SpringBoot 驱动

基于 Spring Boot, 同时提供 starter 组件, 默认集成Druid连接池并配置Undertow作为Web容器,并提供轻量级的建表工具,尽可能的免配置, 快速搭建。

QueryDsl 数据交互

提供方便快捷,类型安全的SQL查询,提供通用的Crud方法,覆盖大部分单表查询场景,不必额外写Dao,只需要关注和编写业务相关的需要关联查询的Dao。

基于 Java 8

设计伊始便基于Java8,以快速开发为目的,希望借助其function,stream,接口的默认与静态方法及lambda表达式等新特性提供方便,简洁,快速的开发体验

<project>
    <parent>
        <groupId>org.huiche</groupId>
        <artifactId>huiche</artifactId>
        <version>${huiche.version}</version>
    </parent>
    <dependencies>
        <dependency>
            <groupId>org.huiche</groupId>

    </dependencies>
